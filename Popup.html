Text-Zusammenfasser Pro

  1. Markiere einen Text auf einer beliebigen Webseite
  2. Klicke mit der rechten Maustaste und wähle "Text zusammenfassen"
  3. Die Zusammenfassung erscheint in einer Seitenleiste
  4. Nutze den Export-Button für Markdown
